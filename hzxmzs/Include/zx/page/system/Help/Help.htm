一、 软件介绍 KingCMS 5.0是集简约的操作界面、灵活的组合方式、无限的扩展能力的ASP网站内容管理系统；自带完善的开发类及函数库；以完全模块化方式设计的框架，不仅是在应用上还是二次开发上，会成为您最得力的开发框架。 二、 目录结构 admin 管理后台所在目录 page 前台系统文件所在目录 template 模板目录 db ACCESS数据库输出目录 up_files 上传文件存储目录 三、 安装说明 下载程序后，设置好IIS，在地址栏里输入路径，如：http://127.0.0.1/admin/ 会自动切换到安装页面，点击第一项进行安装，并再点击第二项删除当前文件。 四、 登录帐号 默认管理员为 admin 密码为 admin888 五、 自定义菜单 自定义菜单是定制后台管理菜单的功能，组合出适合于客户或自己操作习惯的模式，而不是用固定的死板的菜单模式。 六、 管理员设置 可以添加管理员，设置管理员的管理权限和指定某个管理员的菜单、编辑器及语言环境等。 七、 参数设置 网站参数设置。 关键字组：区别于关键字，这是关键字的组合集，添加项目的时候，可以自动调用这个关键字组里的关键字，前提是标题包含这些关键字列表中的关键字。 爬虫：爬虫即指百度、谷歌等搜索引擎来的搜索机器，可以通过这些机器发出的useragent值来判断爬虫类型，可以不用进行修改。 限制IP：因不可避免的会有人对网站进行捣乱或破坏，这个时候需要封掉该用户的操作权限，如封掉IP为111.222.333.444的话，直接写上即可，如果是要封掉一个IP段，则写上前面的部分，如：111.222. 则可以封掉111.222.开头的所有ip了。 八、 模块管理 文章、单页面等模块是需要现在这里进行安装后才可以正常使用。删除不会对数据库进行删除，原先被加入到数据库的数据依然会被保留。 九、 我的帐户 修改密码、修改个性化的管理菜单(自定义菜单里定义的是所有成员的菜单，这里定义的是针对于自己使用的菜单，留空则调用自定义菜单)、指定编辑器和语言包。 十、 退出管理 建议每次操作完成后点击退出来彻底清除cookies，或关掉所有的浏览器。
